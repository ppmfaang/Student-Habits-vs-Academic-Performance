# Student-Habits-vs-Academic-Performance
This project analyzes how students' daily habits, such as study time, sleep patterns, social media use, diet quality, and mental health, affect their academic performance.
The dataset consists of 1,000 simulated student records with multiple lifestyle-related features. The project includes data cleaning, exploratory data analysis (EDA), visualization, and the application of machine learning models to predict students' final exam scores.
The goal of this project is to understand key factors influencing academic performance and build a predictive model based on student habits.
